# 人类玩家功能 - 快速指南

## 🎮 功能概述

现在你可以在阿瓦隆游戏中作为人类玩家参与，与 AI NPC 同台竞技！AI NPC 不会知道你是人类，他们会把你当作另一个 AI 玩家对待。

## ✨ 核心特性

### 1. 身份隐藏
- ✅ AI NPC 不知道你是人类玩家
- ✅ 你的发言自然融入游戏聊天
- ✅ 所有玩家使用统一接口

### 2. 人类玩家功能
- ✅ 文本输入进行发言
- ✅ 参与投票（团队审批、淘汰投票）
- ✅ 任务执行投票
- ✅ 团队提议（当队长时）
- ✅ 特殊角色能力使用

### 3. AI 延迟机制
- ✅ 每位 AI NPC 发言后自动暂停
- ✅ 默认延迟 3 秒（可调整）
- ✅ 给你充足的阅读和反应时间

## 🚀 使用方法

### 启动游戏
```bash
python main.py
```

### 选择是否参与
游戏启动时会询问：
```
============================================================
欢迎来到阿瓦隆游戏！
============================================================

你想要作为人类玩家参与游戏吗？
- 选择 YES：你将与 AI NPC 一起游戏
- 选择 NO：观看 AI NPC 之间的对战
============================================================

请输入 YES 或 NO: 
```

### 游戏流程

#### 1. 角色分配
游戏开始会随机分配你的角色，只有你知道自己的身份：
```
[仅你可见] Human，你的角色是 merlin（梅林）
```

#### 2. 特殊信息
根据角色不同，你会收到私密信息：
- **梅林**：看到邪恶势力名单
- **派西维尔**：看到梅林和莫甘娜（无法区分）
- **邪恶阵营**：看到队友名单

#### 3. 发言阶段
轮到你发言时：
```
[Human] 请输入你的发言：
> 我认为 Player3 的行为很可疑...
```

#### 4. 投票阶段
需要投票时：
```
============================================================
请选择你要投票的对象：
可选对象：Player1, Player2, Player3, Player4, Player5
============================================================

[Human] 你的投票：Player3
```

#### 5. 任务执行
如果你是团队成员：
```
============================================================
你需要对任务结果进行投票
作为好人阵营，你必须投成功票
============================================================

[Human] 请输入 SUCCESS 或 FAIL：SUCCESS
```

## ⚙️ 配置选项

### 调整 AI 延迟
编辑 `main.py` 第 148 行：
```python
ai_delay = 3.0  # 秒
# 建议范围：2.0 - 5.0 秒
# 如果觉得太快，可以调大到 4.0 或 5.0
```

### 玩家数量
- **有人类玩家**: 7 个 AI + 1 个人类 = 8 人局
- **无人类玩家**: 8 个 AI = 8 人局

## 📖 游戏技巧

### 作为好人阵营

#### 梅林 (Merlin)
- 🎯 **目标**: 引导好人获胜，但不暴露身份
- 💡 **技巧**: 
  - 给出暗示但不要过于明显
  - 观察派西维尔的反应
  - 小心莫甘娜的伪装

#### 派西维尔 (Percival)
- 🎯 **目标**: 找出真梅林并保护
- 💡 **技巧**:
  - 观察两个候选人的行为差异
  - 注意谁在提供有用信息
  - 保护你认为的梅林

#### 忠诚仆人 (Loyal Servant)
- 🎯 **目标**: 推理出坏人
- 💡 **技巧**:
  - 仔细分析发言
  - 注意投票模式
  - 相信逻辑而非直觉

### 作为邪恶阵营

#### 刺客 (Assassin)
- 🎯 **目标**: 破坏任务，最后刺杀梅林
- 💡 **技巧**:
  - 隐藏自己直到最后
  - 观察谁是真正的信息提供者
  - 关键时刻一击必杀

#### 莫甘娜 (Morgana)
- 🎯 **目标**: 伪装成梅林迷惑派西维尔
- 💡 **技巧**:
  - 适度表现像梅林
  - 不要过于激进
  - 让派西维尔相信你

#### 莫德雷德 (Mordred)
- 🎯 **目标**: 暗中破坏，对梅林隐藏
- 💡 **技巧**:
  - 利用隐身优势
  - 假装好人
  - 关键时刻投失败票

#### 爪牙 (Minion)
- 🎯 **目标**: 配合队友
- 💡 **技巧**:
  - 混淆视听
  - 支持队友的提议
  - 必要时牺牲自己

## 🎭 输入示例

### 发言输入
```
[Human] 请输入你的发言：
> 我注意到 Player2 在上一轮投票中很积极，这很奇怪
```

### 投票输入
```
[Human] 你的投票：Player2
```

### YES/NO决策
```
[Human] 请输入 YES 或 NO：YES
```

### 团队提议（当队长）
```
作为队长，你需要选择 3 名玩家执行任务
可选玩家：Player1, Player2, Player3, Player4, Player5, Human
请输入玩家名字，用逗号分隔（例如：Player1, Player2, Player3）

[Human] 你的提议：Human, Player1, Player3
```

### 任务投票
```
[Human] 请输入 SUCCESS 或 FAIL：SUCCESS
```

### 最终刺杀（如果是刺客）
```
作为刺客，你需要刺杀一名玩家（如果是梅林，邪恶方获胜）
可选对象：Player1, Player2, Player3, Player4, Player5

[Human] 你的刺杀目标：Player3
```

## ⚠️ 注意事项

### 1. 时间管理
- AI 发言后有 3 秒延迟
- 如果来不及输入，可以按回车跳过当前轮次
- 游戏会自动等待你的输入

### 2. 身份保护
- 不要透露自己是人类
- 保持自然的发言风格
- 像 AI 一样思考和行动

### 3. 游戏平衡
- AI 有延迟，你有充足时间
- 但如果思考太久，其他玩家会等待
- 建议快速但谨慎地决策

## 🔍 常见问题

### Q: AI 会发现我是人类吗？
A: 不会！AI 把你当作另一个 Agent，使用相同的接口交流。

### Q: 我可以调整延迟时间吗？
A: 可以！在 `main.py` 中修改 `ai_delay` 变量。

### Q: 如果我想退出游戏怎么办？
A: 按 Ctrl+C 可以强制退出，但会影响游戏完整性。

### Q: 我能扮演多个角色吗？
A: 不行，每局游戏只能扮演一个角色。

### Q: 人格系统对人类有效吗？
A: 无效，人格系统只影响 AI NPC。

## 📊 游戏统计

### 标准配置
- **玩家数**: 8 人（7 AI + 1 人类 或 8 AI）
- **人格类型**: 10 种 MBTI
- **重复上限**: 每种人格最多 2 个
- **AI 延迟**: 3 秒（可调）

### 推荐设置
- **AI 延迟**: 3.0 - 4.0 秒
- **玩家数**: 7-8 人最佳
- **游戏时长**: 30-45 分钟
