# 阿瓦隆游戏 Web 界面使用指南

## 📋 功能特性

### ✨ 核心功能
- 🎨 **现代化聊天界面** - 类似微信/QQ 的聊天体验
- 🔐 **信息隔离** - 玩家只能看到允许看到的内容
- 📛 **身份持久化** - 身份信息始终显示在界面上方
- 💬 **实时通信** - WebSocket 实现即时消息推送
- 🎭 **NPC 发言过滤** - 只显示 NPC 最终发言，不显示思考过程

### 🎯 界面特点
1. **顶部状态栏**
   - 玩家名称
   - 玩家角色（梅林、派西维尔等）
   - 阵营标识（好人/邪恶）
   - 连接状态

2. **聊天区域**
   -  Moderator 消息（灰色，居中）
   -  NPC 发言（白色气泡，带发言人名称）
   -  私人消息（蓝色背景，左侧边框）
   -  系统消息（斜体，灰色）

3. **输入区域**
   -  文本输入框
   -  发送按钮
   -  清空按钮

## 🚀 快速开始

### 前置要求

确保已安装 Python 3.8+ 和以下依赖：

```bash
pip install aiohttp agentscope ollama
```

或者使用项目的 requirements.txt：

```bash
pip install -r requirements_web.txt
```

### 启动 Web 服务器

#### 方式 1: 默认配置（推荐）
```bash
python web_main.py
```

服务器将在 `http://0.0.0.0:8080` 启动

#### 方式 2: 自定义地址和端口
```bash
python web_main.py 192.168.1.100 9000
```

### 访问游戏

1. 打开浏览器（Chrome、Firefox、Edge 等）
2. 访问显示的地址，例如：`http://localhost:8080`
3. 等待游戏开始

## 🎮 游戏说明

### 界面元素说明

#### 1. 身份面板（顶部）
```
┌─────────────────────────────────────────────┐
│ 👤 玩家：H    📜 角色：梅林    🟦 阵营：好人 │
│                                    🟢 已连接 │

```

- **玩家名称**: 你的游戏内代号（字母 A-H）
- **角色**: 你的身份（梅林、派西维尔等）
- **阵营**: 好人（🟦）或邪恶（🟥）
- **连接状态**: 🟢 已连接 / 🔴 未连接

#### 2. 聊天区域（中间）

**Moderator 消息**（灰色，居中）:
```
        欢迎来到阿瓦隆游戏！等待游戏开始...
```

**NPC 发言**（白色气泡）:
```
A

│ 我认为 B 的发言很有道理... │
└────────────────────────────┘
```

**私人消息**（蓝色背景）:
```
┌──────────────────────────────────┐
│ 📜 你的身份：你的角色是 梅林     │
└──────────────────────────────────┘
```

#### 3. 输入区域（底部）
```
┌─────────────────────────────────────┐
│ [文本输入框...]                     │
│                                     │
│              [清空] [发送]          │
└─────────────────────────────────────┘
```

### 如何参与游戏

1. **发言阶段**
   - 在输入框中输入你的分析和推理
   - 点击"发送"或按 Enter 键提交

2. **投票阶段**
   - 根据提示输入你要投票的对象
   - 例如：输入 "A" 或 "B"

3. **团队提议**
   - 如果你是队长，输入你选择的队员
   - 例如："A, C, H"（选择 3 人团队）

## 🔧 高级配置

### 修改服务器地址

编辑 `web_main.py`:
```python
host = "192.168.3.100"  # 局域网 IP
port = 8080
```

### 启用 HTTPS（生产环境）

需要 SSL 证书：
```python
from aiohttp import web
import ssl

ssl_context = ssl.SSLContext(ssl.PROTOCOL_TLS_SERVER)
ssl_context.load_cert_chain('cert.pem', 'key.pem')

site = web.TCPSite(app, '0.0.0.0', 8443, ssl_context=ssl_context)
```

### 自定义界面样式

编辑 `web_server.py` 中的 CSS 部分：
```python
def get_html_template(self) -> str:

<html>
<head>
    <style>
        /* 在这里修改样式 */
        body { background: your-color; }

...
```

## 🐛 常见问题

### Q: 无法连接到服务器？
**A**: 检查防火墙设置，确保端口 8080 开放

### Q: 界面显示乱码？
**A**: 确保浏览器编码为 UTF-8

### Q: 消息发送失败？
**A**: 检查 WebSocket 连接状态（右上角指示灯）

### Q: 看不到自己的角色？
**A**: 刷新页面或重新连接 WebSocket

## 📝 技术架构

```
┌─────────────┐         ┌──────────────┐         ┌─────────────┐
│   浏览器    │◄───────►│  WebSocket   │◄───────►│  Game Engine│
│  (HTML/CSS) │  WS     │   Server     │  Async  │  (Agents)   │

```

- **前端**: HTML5 + CSS3 + Vanilla JavaScript
- **后端**: Python + aiohttp (异步 Web 框架)
- **通信**: WebSocket (全双工实时通信)
- **游戏引擎**: AgentScope
