## 🎭 阿瓦隆角色速查

### 好人阵营 (Loyal Servants of Arthur)
| 角色 | 能力 | 胜利条件 |
|------|------|----------|
| **梅林** | 开局知道所有坏人（除莫德雷德） | 完成 3 个任务 + 不被刺杀 |
| **派西维尔** | 开局看到梅林和莫甘娜（无法区分） | 保护梅林，完成任务 |
| **忠诚仆人** | 无特殊能力 | 推理、投票、完成任务 |

### 坏人阵营 (Minions of Mordred)
| 角色 | 能力 | 胜利条件 |
|------|------|----------|
| **刺客** | 游戏结束后刺杀一人 | 刺杀梅林 |
| **莫甘娜** | 被派西维尔视为梅林 | 混淆视听，破坏任务 |
| **莫德雷德** | 对梅林隐藏 | 暗中破坏，保护自己 |
| **爪牙** | 无特殊能力 | 伪装、破坏任务 |

## 🎲 游戏流程速记

```
1. 角色分配 → 2. 团队提议 → 3. 投票表决 → 4. 任务执行 → 5. 重复 2-4
                                              ↓
                                    好人完成 3 次成功？
                                              ↓
                                      是 → 最终刺杀
                                              ↓
                                    刺中梅林？→ 坏人胜 / 好人胜
```

## 🐛 常见问题

### Q: 游戏无法启动？
A: 检查 Ollama 服务是否运行：
```bash
ollama list  # 查看可用模型
ollama run qwen3:8b  # 启动模型
```

### Q: 人格未生效？
A: 检查 `personalities/` 目录是否存在且包含 `.md` 文件

### Q: 角色分配不均？
A: 这是正常的，游戏会随机分配角色

### Q: 如何添加新的人格？
A: 在 `personalities/` 目录创建新的 `.md` 文件，格式参考现有文件

## 📊 性能建议

- **推荐玩家数**: 6-8 人（平衡性和流畅度最佳）
- **最小配置**: 5 人（3 好 2 坏）
- **最大配置**: 10 人（7 好 3 坏）
- **内存占用**: ~500MB（8 人局）
- **单局时长**: ~30-60 分钟

## 🎯 获胜技巧

### 好人方
- **梅林**: 给暗示但不暴露，观察派西维尔反应
- **派西维尔**: 仔细分辨真假梅林，保护真梅林
- **忠臣**: 逻辑推理，识别坏人投票模式

### 坏人方
- **刺客**: 暗中观察，找出真梅林
- **莫甘娜**: 适度表现，不要太假或太真
- **莫德雷德**: 利用隐身优势，假装好人
- **爪牙**: 配合队友，制造混乱
