# 🎮 Web 界面游戏启动功能更新

## ✅ 新增功能

### 1. 欢迎面板
- 🎨 精美的欢迎界面
- 📝 清晰的游戏说明
- 🎯 两种游戏模式选择

### 2. 游戏模式选择

#### 🎮 作为玩家加入
- 与 AI NPC 一起游戏
- 体验推理和欺骗的乐趣
- 可以发言、投票、参与任务
- 会分配角色身份

#### 👁️ 作为旁观者  
- 观看 AI NPC 之间的对战
- 不参与游戏决策
- 可以看到所有公开信息

### 3. 开始按钮
- 选择模式后激活
- 点击后进入游戏界面
- WebSocket 实时连接

## 💻 技术实现

### 前端 HTML/CSS/JS
- **欢迎面板**: 独立于游戏界面的 div
- **模式选择**: 卡片式布局，点击选择
- **开始按钮**: 初始禁用，选择模式后激活
- **界面切换**: 隐藏欢迎面板，显示游戏界面

### 后端 API
- **路由**: `POST /api/start_game`
- **参数**: `{ mode: 'player' | 'spectator' }`
- **响应**: 
  ```json
  {
    "success": true,
    "message": "Game started in player mode",
    "mode": "player"
  }
  ```

## 🚀 使用方法

### 1. 启动服务器
```bash
source ~/anaconda3/etc/profile.d/conda.sh
conda activate AgentScope
python web_main.py
```

### 2. 打开浏览器
访问：`http://localhost:8182`

### 3. 选择模式
- 点击"作为玩家加入"或"作为旁观者"卡片
- 卡片会高亮显示选中状态

### 4. 开始游戏
- 点击"🚀 开始游戏"按钮
- 界面切换到游戏聊天界面
- WebSocket 自动连接
