## 🚀 快速启动

### 1. 安装依赖

```bash
pip install aiohttp
```

### 2. 启动 Web 服务器

```bash
python web_main.py
```

### 3. 打开浏览器访问

```
http://localhost:8080
```

## 🎯 使用说明

### 人类玩家
1. 打开浏览器访问服务器地址
2. 等待游戏开始
3. 查看自己的身份（显示在顶部）
4. 在讨论阶段输入发言
5. 在投票阶段输入投票对象
6. 在团队提议阶段输入选择的队员

### 信息隔离规则

✅ **玩家可以看到**:
- 公开的游戏消息（Moderator 广播）
- 所有 NPC 的最终发言
- 自己的私人身份信息
- 与自己的交互请求

❌ **玩家看不到**:
- NPC 的思考过程
- 其他玩家的私人信息
- 邪恶势力的秘密讨论
- 梅林看到的邪恶玩家名单

## 🔧 技术细节

### WebSocket 消息类型

```python
# 游戏开始
{"type": "game_start"}

# 角色揭示（私人）
{"type": "role_reveal_private", "role": "merlin", "team": "good"}

# 讨论发言
{"type": "discussion", "speaker": "A", "content": "..."}

# 投票结果
{"type": "vote_result", "content": "YES: 5, NO: 2"}

# 任务结果
{"type": "quest_result", "content": "成功"}

# 游戏结束
{"type": "game_end", "content": "好人获胜"}
```

### 消息过滤逻辑

```python
# 公开消息（所有人可见）
public_types = [
    "game_start", 
    "role_reveal_public", 
    "discussion", 
    "vote_result", 
    "quest_result", 
    "game_end"
]

# 私人消息（仅人类玩家可见）
private_types = [
    "role_reveal_private",
    "input_request"
]
```
