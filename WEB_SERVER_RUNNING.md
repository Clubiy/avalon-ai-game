# 🌐 Web 界面启动说明

## ✅ 当前状态

Web 服务器已经可以正常运行！

## 🚀 启动方法

### 方式 1: 使用 AgentScope 环境（推荐）

```bash
source ~/anaconda3/etc/profile.d/conda.sh
conda activate AgentScope
python web_main.py
```

### 方式 2: 如果已安装 aiohttp

```bash
pip install aiohttp
python web_main.py
```

## 📡 访问地址

启动后会显示：
```
============================================================

============================================================

🌐 请在浏览器中打开上述地址开始游戏
```

在浏览器中访问：**http://localhost:8182**

## 🔧 自定义配置

### 修改端口
编辑 `web_main.py` 第 44 行：
```python
port = 8182  # 改成你想要的端口
```

### 修改监听地址
编辑 `web_main.py` 第 43 行：
```python
host = "0.0.0.0"  # 监听所有网络接口
# 或
host = "127.0.0.1"  # 只允许本地访问
```

## 📝 注意事项

⚠️ **重要提示**:
1. 确保防火墙允许对应端口的连接
2. 如果在远程服务器上运行，需要开放相应端口
3. 默认端口是 8182（不是 8080）

## 🐛 常见问题

### Q: 无法访问网页？
**A**: 
- 检查防火墙设置
- 确认端口未被占用
- 如果是远程服务器，确保安全组开放该端口

### Q: 显示"未连接"？
**A**: 
- WebSocket 会自动重连
- 刷新页面
- 检查服务器是否正常运行

### Q: 看不到自己的角色？
**A**: 
- 这是正常的，因为游戏还没开始
- 目前只是前端界面，还需要与游戏引擎集成
